# Eureka Discovery Service
### Technologien
- Spring Boot
- Netflix Eureka Server
- Spring Cloud

## Konfiguration
- **Port**: `8761`
- **Service Name**: `eureka_server`

## Endpunkte
- **Dashboard**: `http://localhost:8761`
- **REST API**: `http://localhost:8761/eureka`

1. **game-service** (Port 8081)
2. **board-service** (Port 8083)
